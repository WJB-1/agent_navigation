🛠️ 解决方案

高德地图 JSAPI 2.0 需要配置安全密钥！
自 2021 年 12 月起，高德地图要求所有 Key 必须配置安全密钥才能正常使用。

解决步骤：

1
登录高德控制台
访问 https://console.amap.com/dev/key/app
使用您的高德账号登录
2
找到对应的应用和 Key
在「应用管理」→「我的应用」中找到您正在使用的 Key
当前使用的 Key: 0beb99f14b04f13a734c81c032b9a8f1
3
配置安全密钥
点击「设置」→「安全密钥」
生成一个新的安全密钥，或者使用已有的
// 在加载地图前设置安全密钥 window._AMapSecurityConfig = { securityJsCode: '您的安全密钥', };
4
配置域名白名单
在 Key 的「域名白名单」中添加：
  • localhost (本地开发)
  • 127.0.0.1 (本地 IP)
  • 您的实际域名 (如 yourdomain.com)
5

错误信息 原因 解决方案
USERKEY_PLAT_NOMATCH Key 与平台不匹配 检查 Key 是否配置了 Web 服务
